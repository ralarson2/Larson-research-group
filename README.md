# Larson-research-group
Research webpage for Professor Rebecca Larson's research group at the University of Wisconsin-Madison

This repository hosts the static site for the Larson Research Group. 
The site provides information about current research areas, publications, 
students, and resources. It also includes a section that displays live air 
quality data from the Uganda sensor network API.

To update the site, edit the HTML files in this repository and commit the 
changes. GitHub Pages will automatically rebuild and publish the site.
